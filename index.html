Welcome to SwingBot

GitHub Pages serves the nightly export from the data/ folder. Use these links in your Custom GPT action configuration:

  • data/signals.json – machine-readable signals & positions
  • data/highlights.md – one-glance human brief
Heads up: The GitHub Actions workflow updates these files every weekday ~5:05 pm ET. If you’re testing locally, run python run.py before pushing so Pages reflects the latest snapshot.
